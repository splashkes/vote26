Installation Instructions

For WordPress Installation:

  1. Copy the art-battle-external/wordpress-plugin/ folder to your WordPress /wp-content/plugins/ directory
  2. Rename the folder to art-battle-artists-display
  3. Activate the plugin in WordPress admin under Plugins
  4. Use the shortcode [art-battle-artists event="AB3333"] in any post or page

Shortcode Examples:

  • [art-battle-artists event="AB3053"] - Basic grid layout
  • [art-battle-artists event="AB2900" layout="list"] - List layout
  • [art-battle-artists event="AB3026" show_images="no"] - No images
  • [art-battle-artists event="AB3053" layout="list" show_bios="no"] - List without bios
